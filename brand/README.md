# Brand

Pulled from PRD §9. Settled, not up for v1 debate.

## Wordmark

"prism" lowercase, **Geist Black (900)**, three layers with `mix-blend-mode: screen`:

- Magenta ghost `#FF1F8E` at 55% opacity, shifted -2px on X
- Cyan ghost `#1FE6FF` at 55% opacity, shifted +2px on X
- White foreground `#FAFAF9`

Letter-spacing `-0.04em`. Reference: [`wordmark-dark.html`](wordmark-dark.html).

## App icon

Lowercase "p" in the same chromatic aberration treatment, on near-black `#0F0F10` tile, rounded square per platform conventions. Pending Claude Design final.

## Aberration usage

Wordmark and app icon only. UI icons stay clean (Lucide, 1.5px stroke).

## Substrate colors

- Background: `#0A0A0B`
- Foreground: `#FAFAF9`
- Icon tile: `#0F0F10`

## Typography

- Geist Sans — UI, body
- Geist Mono — code, terminal, agent logs

OFL licensed, free.

## Voice

Direct, utilitarian, dry humor allowed. No emoji in product. No exclamation points.
